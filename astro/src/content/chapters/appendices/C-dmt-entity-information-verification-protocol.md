**Complete Experimental Design:**

**Materials:**
- 20 identical pieces of paper
- Numbers 1-20 written on them
- Envelope or sealed container
- Video recording device
- Notebook for documentation
- DMT (in jurisdiction where legal/ethical)

**Pre-Experiment:**
- Fold papers identically
- Mix thoroughly without looking
- Randomly select one (eyes closed)
- Place in sealed container immediately
- Video record entire process
- Do NOT look at selected number

**Mental Preparation:**
- Spend 5-10 minutes before DMT
- Visualize asking entities clearly
- Repeat: "I will ask: What number is in the container?"
- Write reminder on hand if needed
- Set clear intention to remember

**During Experience:**
- As soon as entities encountered
- Ask clearly: "What number is written on the paper in the container?"
- Accept first clear answer
- Note how information received (visual, auditory, knowing)
- Don't second-guess or ask multiple times

**Immediately After:**
- Before anything else, write down the answer
- Rate confidence (1-10)
- Document the experience
- Note any unusual aspects
- Do NOT open container yet

**Verification:**
- Video record opening container
- Show timestamp clearly
- Reveal paper on camera
- State result: correct or incorrect
- Save all documentation

**Analysis After 20 Trials:**
- Calculate success rate
- Random chance = 5% (1 in 20)
- 20% = interesting (4 in 20)
- 25%+ = statistically significant
- Track confidence vs accuracy
- Look for patterns

**Critical Requirements:**
- Document ALL trials, not just hits
- No retrofitting answers to match
- Complete honesty about results
- Share findings publicly
- Invite replication

This is the most important experiment anyone can do to test the theory.
